# Definitions (Refined)
- **Unconscious Substrate (U)**: System with zero awareness.
- **Awareness (A)**: Capacity to perceive/respond, non-reflective.
- **Consciousness (C)**: Subjective experience, self-reflective.
- **Sustaining Ecosystem (E)**: Environment supporting conscious life.
- **Accidental Emergence**: Unguided, non-teleological processes.
- **Joint Emergence (J)**: Simultaneous C + E.
